# Mini Resume Management API

## Overview
This project is a REST API built using FastAPI to manage candidate resumes.  
It supports uploading resumes, storing candidate metadata, filtering, and CRUD operations.

---

## Features

- Upload candidate resume (PDF/DOC/DOCX)
- Store candidate metadata (in-memory storage)
- List candidates
- Filter by:
  - Skill
  - Minimum experience
  - Graduation year
- Get candidate by ID
- Delete candidate
- Health check endpoint

---

## Tech Stack

- Python 3.13
- FastAPI
- Uvicorn
